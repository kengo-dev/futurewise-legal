# FutureWise Legal Documents

このリポジトリは、iOS アプリ **FutureWise** に関する法務文書を公開するためのものです。  
GitHub Pages を通じて一般公開しています。

---

## 📑 公開中のドキュメント

- [利用規約 (Terms of Service)](https://kengo-dev.github.io/futurewise-legal/terms)  
- [プライバシーポリシー (Privacy Policy)](https://kengo-dev.github.io/futurewise-legal/privacy)
- [サポート (Support)](https://kengo-dev.github.io/futurewise-legal/support)

---

## ℹ️ 目的
App Store 審査およびユーザーがアプリ利用前に確認できるよう、  
利用規約とプライバシーポリシーをオンラインで公開しています。  

---

## 🛠 運用
- ドキュメントは Markdown (`terms.md`, `privacy.md`, `support.md`) で管理しています。
- `main` ブランチに push された内容が自動的に GitHub Pages に反映されます。
- 必要に応じて文言の更新を行い、更新日時を明記してください。
